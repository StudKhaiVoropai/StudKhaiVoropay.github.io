Завдання 1 Теорія

Scilab - це вільне та відкрите середовище чисельних обчислень, яке надає широкі можливості для математичного моделювання, аналізу даних, розв'язання рівнянь, візуалізації та багато іншого. Він базується на мові програмування Scilab, яка має синтаксис, схожий на MATLAB.
Scilab використовується в багатьох областях, включаючи наукові дослідження, інженерію, фінанси, статистику, оптимізацію та багато іншого. Він надає потужні математичні функції та інструменти для обчислення, моделювання та аналізу даних. Scilab також має широку спільноту користувачів і документацію, що дозволяє використовувати його для вирішення різноманітних завдань.
Основні функціональні можливості Scilab:
Розв'язання лінійних та нелінійних рівнянь;
Чисельна оптимізація та побудова графіків;
Аналіз даних та статистика;
Матричні операції та лінійна алгебра;
Фізичні обчислення та моделювання;
Керування системами та сигналами;
Символьні обчислення;
Інтегрування з іншими мовами програмування, такими як C, Fortran, Python та іншими.
Scilab є потужним інструментом для розв'язання різноманітних математичних задач, від простих до складних. Він надає зручний інтерфейс програмування, багатий функціонал та широкі можливості візуалізації результатів. Використовуючи Scilab, ви можете вирішувати математичні завдання, моделювати системи, аналізувати дані та багато іншого.
Для ознайомлення з роботою в Scilab було надано три приклада:
1) Приклад з побудовою графіку функції: Перший приклад показує використання Scilab для побудови графіка функції. Ми визначили функцію f(x), обчислили значення функції на заданому проміжку значень x і використали функцію plot (або plot2d) для побудови графіка. Додаткові команди використовуються для підписів осей та візуального налаштування графіка.
2) Приклад з чисельним розв'язанням рівняння: В другому прикладі використовувалася функція fsolve для чисельного розв'язання рівняння. Ця функція дозволяє знайти корені нелінійних рівнянь шляхом чисельного ітеративного методу. Ми використали цю функцію, щоб знайти корені рівняння і вивести їх на екран.
3) Приклад з чисельним розв'язанням системи рівнянь: Для вирішення системи рівнянь потрібно спочатку визначити функції, що відповідають кожному рівнянню системи. Далі, ми визначаємо функції update_x та update_y, які обчислюють нові значення x та y на кожній ітерації.
Основним завданням вирішення системи рівнянь є знаходження таких значень x та y, при яких обидва рівняння стають справедливими одночасно. Чисельні методи, такі як метод Ньютона, використовуються для знаходження розв'язків системи рівнянь шляхом ітеративного оновлення значень x та y до досягнення збіжності.
Для перевірки того що рішення в Scilab працюють коректно, їх тагож буде потрібно перевірити за допомогою excel.
Microsoft Excel - це потужна електронна таблиця, яка входить до складу пакета програм Microsoft Office. Вона широко використовується для роботи з числовими даними, виконання розрахунків, аналізу даних та створення графіків і діаграм.
Excel може бути корисним для створення звітів, аналізу даних, управління проектами, фінансового моделювання та багатьох інших завдань, де потрібно обробляти числові дані та виконувати розрахунки.
